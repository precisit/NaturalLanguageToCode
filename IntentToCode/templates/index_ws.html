Instructions to code

Feel free to give instructions! Write or say (click the microphone icon) what you want the different objects to do. For example "The character should cut the tree with the axe". Or paraphrased: "The boy should cut the bush with the hatchet". The enviroment specified for this experiment consist of a number of object and possible actions. These are:

Actions:
  • Hit (*target*, *with*)
  • Walk (*direction*)
  • Jump (*how*)
  • Wave ()
  • Cut (*target*, *with*)
  • Eat ()
  • Turn (*direction*)
 Objects:
  • Character
  • Tree
  • Axe
  • Cow
  • Johanna
