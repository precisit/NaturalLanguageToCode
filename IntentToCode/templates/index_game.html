Instructions to code

Feel free to give instructions! Write what you want the different objects to do. For example "The character should cut the tree with the axe". Or using synonyms: "The boy should cut the bush with the hatchet". Your goal is to move the character to the goal. Remember to be precise. A computer can have problems with ambiguous words like 'left' and 'use'.

The enviroment specified for this experiment consist of a number of objects and possible actions. These are:

Objects:

  • Axe (Item)
  • Character
  • Cow
  • Tree
Actions:
  • Walk (*direction* / *target*) - Walking one step in specified direction or to the target
  • Jump (*direction*)
    • - Jumping two steps in specified direction
  • Cut (*target*, *with*)
    • - Cuting target with specified item, destroying the target.
  • Eat (*target*)
    • - Eats the target, destroying it.
